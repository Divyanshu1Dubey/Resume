# Divyanshu Dubey — Portfolio 🚀

A clean, responsive personal portfolio website built with React.

## Sections 📚

- About Me & Skills
- Education
- Experience
- Projects & Publications
- Freelance (Tech4Growth)
- Contact

## Getting Started

```bash
npm install
npm start
```

## Build & Deploy

```bash
npm run build
```

Deploy the `build/` folder to any static hosting (GitHub Pages, Netlify, Vercel, etc.).

## Customization

Edit `src/portfolio.js` to update all personal information, skills, experience, education, projects, and contact details.

## Technologies Used 🛠️

- [React](https://reactjs.org/)
- [React Router](https://reactrouter.com/)
- [react-reveal](https://www.react-reveal.com/)
- [baseui](https://github.com/uber/baseweb)
